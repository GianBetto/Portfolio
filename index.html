Chi Sono

Completano il mio background conoscenze di economia aziendale e contabilità, acquisite presso l'Istituto Belotti di Bergamo, utili per operare in contesti con visione economica e organizzativa.

Sono una persona affidabile, responsabile e precisa, che affronta gli impegni con serietà e attenzione ai dettagli. Collaborativo e disponibile, offro supporto concreto e contribuisco attivamente al raggiungimento degli obiettivi del team.

Appassionato di sport, tecnologia e innovazione, coltivo hobby che riflettono curiosità, manualità e spirito pratico: suono nella Fanfara di Scanzorosciate, pratico corsa, ciclismo, nuoto e sci, e seguo MotoGP e Formula 1. Amo smontare, analizzare e riparare oggetti per comprenderne il funzionamento e trovare soluzioni pratiche. Tra i miei interessi personali rientrano le nuove tecnologie, l’informatica, la finanza personale e i viaggi, che mantengono viva la mia curiosità e apertura mentale.

I Miei Progetti

Yogatella: Sito web di Yoga

Progettazione di pagina web relativa a corsi di yoga, eventi, insegnanti e contatti del centro. L’interfaccia è stata sviluppata seguendo le euristiche di Nielsen e il modello MiLe, con particolare attenzione all’accessibilità e alla facilità di navigazione, garantendo un’esperienza coerente e intuitiva su diversi dispositivi.

Vue Nuxt Node MongoDB GitHub Vercel

GeoAir: dashboard web per la qualità dell'aria mediante API

GeoAir è una piattaforma web avanzata per l’analisi e la visualizzazione dei dati sulla qualità dell’aria nella regione Lombardia. Sviluppata con tecnologie web moderne e basata su un’infrastruttura che integra Python per l’elaborazione dei dati e API per l’accesso e l’aggiornamento in tempo reale delle informazioni ambientali, la piattaforma offre mappe interattive, analisi delle tendenze e dashboard dinamiche. GeoAir supporta enti pubblici, ricercatori e decisori nella valutazione dell’inquinamento atmosferico e nella pianificazione di politiche ambientali basate su dati concreti.

Python Flask API GitHub
GitHub

GIS project: Air Quality Mapping

Progetto che studia l’evoluzione storica di NO₂, PM₂.₅ e PM₁₀ in Germania, correlando i livelli di inquinamento con la copertura del suolo e la densità di popolazione. Include un’analisi temporale (2013–2022) e una valutazione dell’esposizione della popolazione rispetto alle linee guida UE e OMS, visualizzate tramite una WebGIS interattiva.

HTML CSS JavaScript QGIS GeoServer GitHub
Demo Live GitHub

IDRONET: mitigazione del rischio idraulico

Progetto di gruppo volto alla mitigazione del rischio idraulico nel bacino Seveso–Lambro–Olona attraverso un sistema integrato di vasche di laminazione e aree umide di fitodepurazione naturale, con l’obiettivo di ridurre le esondazioni, migliorare la qualità delle acque e valorizzare le funzioni ecologiche e pubbliche del territorio.

Prot3ct: E-Commerce disinfettanti

Progettazione di una piattaforma e-commerce per la vendita di prodotti disinfettanti. Ecommerce completo con gestione prodotti, carrello, pagamenti e autenticazione utenti. Sistema di amministrazione per la gestione del catalogo e degli ordini mediante gestione database integrata nella webapp.
